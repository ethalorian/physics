## 📍 How to Use It

### **Creating Homework:**
1. Click "Create New" button
2. Select "Homework Assignment" tab
3. Click "Go to Assignment Builder"
4. Build your assignment
5. Save & Publish
6. Returns to Homework Library tab

### **Assigning to Students:**
1. Click "Create New" button
2. Select "Assign Lesson" tab
3. Fill in form (lesson, class, due date)
4. Submit
5. Appears in "Assigned to Students" tab

### **Grading:**
1. Go to "Results & Grading" tab
2. See all submissions
3. Click "Review" on submission
4. Grade and provide feedback
5. Student sees results

## 📊 Information Architecture

### **Homework Library Tab:**
- **Purpose:** Create and manage homework content
- **Shows:** Individual assignments with questions
- **Actions:** Create, edit, delete, preview
- **Data source:** AssignmentContext

### **Assigned to Students Tab:**
- **Purpose:** Manage class assignments
- **Shows:** What's assigned to which classes
- **Actions:** Assign, remove, track progress
- **Data source:** AssignmentSystemContext

### **Results & Grading Tab:**
- **Purpose:** Review student work
- **Shows:** Submissions and grades
- **Actions:** Grade, provide feedback, export
- **Data source:** Submissions from both contexts
