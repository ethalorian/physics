# Unit 1 content blocks

Unit 1 lessons (slugs `u1-d01`…`u1-d22`) are authored as `content_blocks` JSON on
the `lessons` rows in Supabase (project PhysicsAPP). This folder is a snapshot for
version control / review.

  scaffold / physics-diagram background. Used wherever a prompt intimates a
  student DRAWING (position–time & velocity–time graphs, tip-to-tail vectors,
  free-body diagrams).
- Draw-intent blocks converted from text `exit_ticket` / generic `doodle` to
  `sketch`: d02 b6, d03 b10, d04 b7, d06 b8, d10 b8, d14 b8, d15 b8 (+ d15 b10
  reworded to the written check).
- Days 19–22 authored fresh (Workshop, Synthesis, Transfer×2). The Day 21–22
  transfer task is completed ON PAPER in class; the online pages are
  planning/reflection scaffolds around it.
- In-class-only labs (Vernier match-the-graph, cart/track) keep their sim
  substitutes (race-track, maze-vectors, etc.) + data tables, so an at-home
  student gets an equivalent investigation.
