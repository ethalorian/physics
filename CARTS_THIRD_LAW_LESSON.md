# Carts & Springs: Newton's Third Law - Complete Lesson

## 🎯 Overview

The **classic cart problem** is one of the best demonstrations of Newton's Third Law. Two carts sit on a frictionless track with a compressed spring between them. When released, the spring pushes the carts apart with equal and opposite forces, demonstrating action-reaction pairs and conservation of momentum.

## 📚 Learning Objectives

By the end of this lesson, students will be able to:

1. **State Newton's Third Law** accurately
2. **Identify action-reaction pairs** in physical situations
3. **Explain** why forces are equal and opposite
4. **Predict** motion when forces act on different masses
5. **Apply F = ma** to both objects simultaneously
6. **Calculate velocities** using momentum conservation
7. **Verify** that total momentum remains constant
8. **Distinguish** between internal and external forces

## ⚖️ Newton's Third Law

### Statement

**"For every action force, there is an equal and opposite reaction force."**

Or more precisely:

**"When object A exerts a force on object B, object B simultaneously exerts a force on object A that is equal in magnitude and opposite in direction."**

### Mathematical Form

\[
\vec{F}_{A \text{ on } B} = -\vec{F}_{B \text{ on } A}
\]

The forces are:
- **Equal in magnitude**: \( |F_{A \text{ on } B}| = |F_{B \text{ on } A}| \)
- **Opposite in direction**: One positive, one negative
- **Act on different objects**: One on A, one on B
- **Occur simultaneously**: Not cause and effect!

## 🚂 The Classic Cart Problem

### Physical Setup

**Equipment:**
- **Two carts** on a frictionless track
- **Compressed spring** between them
- **Release mechanism** to let spring push them apart

**Initial Conditions:**
- Both carts at rest (v = 0)
- Spring compressed (stores energy)
- System isolated (no external forces)

**What Happens:**
1. Spring is released
2. Spring pushes Cart A to the left
3. Spring pushes Cart B to the right
4. Forces are **equal and opposite**
5. Carts accelerate in opposite directions
6. After spring extends, carts continue at constant velocities

### Key Observations

**Same Force, Different Effects:**
- **If masses are equal**: Carts move at same speed, opposite directions
- **If masses are different**: Lighter cart moves faster!

**Why?** Newton's Second Law: \( F = ma \), so \( a = \frac{F}{m} \)
- Same F, smaller m → bigger a
- Same F, bigger m → smaller a

## 🔬 Physics Analysis

### Forces (Newton's 3rd Law)

The spring exerts:
- **Force ON Cart A**: \( F_A = +F \) (pushing right, let's say)
- **Force ON Cart B**: \( F_B = -F \) (pushing left)

**Key Points:**
- These are **equal in magnitude**: \( |F_A| = |F_B| \)
- These are **opposite in direction**: opposite signs
- These act **simultaneously**: not sequential
- These act on **different objects**: one on A, one on B

### Accelerations (Newton's 2nd Law)

Each cart accelerates according to F = ma:

\[
a_A = \frac{F_A}{m_A} = \frac{F}{m_A}
\]
\[
a_B = \frac{F_B}{m_B} = \frac{-F}{m_B}
\]

**Example:**
- Spring force = 100 N
- Cart A: m = 2 kg → a = 100/2 = 50 m/s² (right)
- Cart B: m = 1 kg → a = -100/1 = -100 m/s² (left)

Notice: Lighter cart (B) has **twice the acceleration**!

### Velocities

After time t:
\[
v_A = v_{0A} + a_A t = 0 + \frac{F}{m_A} t
\]
\[
v_B = v_{0B} + a_B t = 0 - \frac{F}{m_B} t
\]

### Momentum Conservation

**Initial momentum** (both at rest):
\[
p_{initial} = m_A v_{0A} + m_B v_{0B} = 0 + 0 = 0
\]

**Final momentum** (after pushing apart):
\[
p_{final} = m_A v_A + m_B v_B
\]

**Conservation law** (no external forces):
\[
p_{final} = p_{initial} = 0
\]

Therefore:
\[
m_A v_A + m_B v_B = 0
\]
\[
m_A v_A = -m_B v_B
\]

This means: **Momenta are equal and opposite!**

## 📊 Sample Calculations

### Example 1: Equal Masses

**Given:**
- m<sub>A</sub> = 2.0 kg
- m<sub>B</sub> = 2.0 kg  
- Spring force = 100 N for 0.5 seconds

**Calculate accelerations:**
- a<sub>A</sub> = F/m<sub>A</sub> = 100/2 = 50 m/s²
- a<sub>B</sub> = -F/m<sub>B</sub> = -100/2 = -50 m/s²

**Calculate final velocities:**
- v<sub>A</sub> = 0 + 50(0.5) = **25 m/s** (right)
- v<sub>B</sub> = 0 - 50(0.5) = **-25 m/s** (left)

**Verify momentum conservation:**
- p<sub>total</sub> = 2(25) + 2(-25) = 50 - 50 = **0 ✓**

### Example 2: Different Masses (2:1 ratio)

**Given:**
- m<sub>A</sub> = 2.0 kg
- m<sub>B</sub> = 1.0 kg
- Spring force = 100 N for 0.5 seconds

**Calculate accelerations:**
- a<sub>A</sub> = 100/2 = 50 m/s²
- a<sub>B</sub> = -100/1 = -100 m/s² (twice as much!)

**Calculate final velocities:**
- v<sub>A</sub> = 0 + 50(0.5) = **25 m/s** (right)
- v<sub>B</sub> = 0 - 100(0.5) = **-50 m/s** (left, twice as fast!)

**Verify momentum conservation:**
- p<sub>total</sub> = 2(25) + 1(-50) = 50 - 50 = **0 ✓**

**Key Insight:** Lighter cart (m<sub>B</sub>) moves **twice as fast** as heavier cart (m<sub>A</sub>)!

### Example 3: Calculate Unknown Velocity

**Given:**
- m<sub>A</sub> = 3.0 kg, v<sub>A</sub> = 10 m/s (after push)
- m<sub>B</sub> = 2.0 kg, v<sub>B</sub> = ? (unknown)
- Started from rest

**Using momentum conservation:**
\[
m_A v_A + m_B v_B = 0
\]
\[
3(10) + 2(v_B) = 0
\]
\[
30 + 2v_B = 0
\]
\[
v_B = -15 \text{ m/s}
\]

**Answer:** Cart B moves at **15 m/s in the opposite direction**

## 🧮 Key Mathematical Relationships

### From Forces to Velocities

**Step 1:** Apply Newton's 3rd Law
\[
F_{\text{on A}} = -F_{\text{on B}} = F
\]

**Step 2:** Apply Newton's 2nd Law to each cart
\[
a_A = \frac{F}{m_A}, \quad a_B = \frac{-F}{m_B}
\]

**Step 3:** Calculate velocities (starting from rest)
\[
v_A = a_A t = \frac{F t}{m_A}
\]
\[
v_B = a_B t = \frac{-F t}{m_B}
\]

**Step 4:** Verify momentum conservation
\[
p_{total} = m_A v_A + m_B v_B = m_A \left(\frac{Ft}{m_A}\right) + m_B \left(\frac{-Ft}{m_B}\right) = Ft - Ft = 0 \,\checkmark
\]

### Velocity Ratio from Mass Ratio

From momentum conservation (\( m_A v_A = -m_B v_B \)):
\[
\frac{v_A}{v_B} = -\frac{m_B}{m_A}
\]

**Example:**
- If m<sub>A</sub> = 3 kg and m<sub>B</sub> = 1 kg
- Then v<sub>A</sub>/v<sub>B</sub> = -1/3
- **Lighter cart moves 3× faster!**

### Numerical Problems

**Problem 1:**
Two carts with masses m<sub>A</sub> = 2.0 kg and m<sub>B</sub> = 1.0 kg are pushed apart by a 150 N spring for 0.4 seconds.

a) Calculate the acceleration of each cart  
b) Calculate the final velocity of each cart  
c) Verify momentum conservation

**Answers:**
```
a) a_A = F/m_A = 150/2.0 = 75 m/s²
   a_B = F/m_B = 150/1.0 = 150 m/s²

b) v_A = a_A × t = 75 × 0.4 = 30 m/s
   v_B = -a_B × t = -150 × 0.4 = -60 m/s

c) p_total = m_A v_A + m_B v_B
           = 2.0(30) + 1.0(-60)
           = 60 - 60 = 0 ✓
```

**Problem 2:**
Cart A (3 kg) moves at 12 m/s after the push. What is Cart B's velocity if it has mass 2 kg?

**Solution using momentum conservation:**
```
m_A v_A + m_B v_B = 0
3(12) + 2(v_B) = 0
36 + 2v_B = 0
v_B = -18 m/s
```

**Problem 3:**
The spring exerts 200 N on each cart for 0.3 seconds. If Cart A ends up moving at 20 m/s, what is its mass?

**Solution:**
```
v_A = (F/m_A) × t
20 = (200/m_A) × 0.3
20 = 60/m_A
m_A = 60/20 = 3.0 kg
```

## 🎯 Common Misconceptions

### Misconception 1: "Heavier objects exert more force"

**Reality:** Forces in an action-reaction pair are **always equal**, regardless of mass.

**Demonstration:**
- Set m<sub>A</sub> = 5 kg, m<sub>B</sub> = 1 kg
- Watch force arrows during push
- Both show same magnitude (e.g., 100 N)
- Forces are equal even though masses differ!

### Misconception 2: "Action causes reaction"

**Reality:** Action and reaction occur **simultaneously**, not sequentially.

**Clarification:**
- Not cause-and-effect
- Both forces exist at the same moment
- Can't have one without the other
- They're two aspects of the same interaction

### Misconception 3: "Forces cancel out"

**Reality:** Forces act on **different objects**, so they don't cancel.

**Explanation:**
- Force on A affects A's motion
- Force on B affects B's motion
- Can't cancel forces on different objects
- Each cart accelerates according to the force on it

### Misconception 4: "Equal forces → equal velocities"

**Reality:** Equal forces → equal **acceleration changes**, not equal velocities.

**Demonstration:**
- With F = ma, same F but different m gives different a
- Different a over same time → different Δv
- Example: a<sub>light</sub> = 2 × a<sub>heavy</sub> → v<sub>light</sub> = 2 × v<sub>heavy</sub>

## 📐 Advanced: Center of Mass

**Bonus Concept:**

The center of mass doesn't move! 

\[
x_{cm} = \frac{m_A x_A + m_B x_B}{m_A + m_B}
\]

Since no external forces act, the center of mass stays at the same position throughout.
